# OctoCut

OctoCut은 .NET 10 WPF로 만든 간단한 동영상 편집 도구입니다. 하나의 영상을 열고, 타임라인에서 구간을 나눈 뒤, 필요 없는 클립을 제거하고 FFmpeg로 결과물을 렌더링하는 MVP를 목표로 합니다.

## 구현된 MVP 기능

- 단일 비디오 파일 열기
- 현재 재생 위치 기준으로 클립 분할
- 분할된 클립 삭제
- FFmpeg 렌더링
  - 무재인코딩 모드: 가능한 경우 빠르게 스트림 복사
  - 인코딩 모드: FFmpeg로 새 출력 파일 생성
- FFmpeg를 찾을 수 없을 때 안내 팝업 표시
- FFmpeg 실행 파일 경로 수동 지정
- `File > Settings` 메뉴에서 설정 열기

## 사전 준비

- .NET 10 SDK
- FFmpeg
  - 앱 실행 자체에는 필수가 아닐 수 있지만, 렌더링에는 필요합니다.
  - 시스템 `PATH`에 등록하거나 앱 설정에서 `ffmpeg.exe` 경로를 직접 지정하세요.

## 기본 사용법

1. OctoCut을 실행합니다.
2. 비디오 파일을 엽니다.
3. 원하는 위치로 이동한 뒤 `S` 키 또는 분할 버튼으로 클립을 나눕니다.
4. 제거할 분할 클립을 삭제합니다.
5. 렌더링 모드를 선택하고 출력 파일을 생성합니다.

## 렌더링 동작

- 무재인코딩 모드는 FFmpeg의 스트림 복사를 사용하므로 빠르지만, 원본 포맷과 컷 지점에 따라 정확도나 호환성에 제한이 있을 수 있습니다.
- 인코딩 모드는 출력 호환성이 더 안정적일 수 있지만 시간이 더 걸립니다.
- FFmpeg가 설치되어 있지 않거나 경로를 찾을 수 없으면 팝업으로 안내하며, 설정에서 수동 경로를 지정할 수 있습니다.
